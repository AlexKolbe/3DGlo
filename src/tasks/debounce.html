Статус: Ожидание ввода... Таймер: 0 мс

Как это работает:

  • Каждый введённый символ запускает новый таймер на 300 мс
  • Предыдущий таймер сбрасывается при новом вводе
  • Текст обновляется только после паузы в 300 мс
  • Это полезно для оптимизации частых обновлений UI
